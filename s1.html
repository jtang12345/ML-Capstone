Session 1: Introduction to Machine Learning

What is Machine Learning?

  • Machine learning helps computers learn like humans, but more efficiently.
  • As machines go through the process of learning by processing data and looking for patterns, it can then make decisions and predictions based on what was learned and continue to adapt and refine its models as more and more data is used.

Influences of Machine Learning

  • Machines have the ability to process lots of data at a time.
  • This makes them super efficient and provides a higher potential for accuracy.
  • Machines can process data with more accuracy and can be trained to filter out data to narrow results down
  • When there is more accuracy, we get better results
    • Medicine: For example: using data from a variety of symptoms and the diagnosis for those range of symptoms, can make the diagnosis process more accurate and make sure we are as healthy as we can be.

Components of Machine Learning

Representation

  • Various machine learning models can be used to represent data by finding patterns
    • Bayesian networks
    • Neural networks

Evaluation

  • With so many different types of models, evaluation is used to determine which model works best for the data

Optimization

  • Since there are many ways to configure a model, there are several ways to optimize the model.

Types of Machine Learning

Supervised Learning

  • Data is labeled. Less work for machines but more work for humans especially when dealing with lots of data.
    • Labels: data that is described to the machine
  • Looking for a specific output
  • Also known as inductive learning

Unsupervised Learning

  • Data is not labeled. Machines look for patterns with clustering without supervision
  • Output is not necessarily what you want and can be completely random

Semi-Supervised Learning

  • Mixture of supervised and unsupervised learning
  • Combines the unsupervised aspect of clustering unlabeled data and supervised learning by accurately labeling some of the unlabeled dataset. Then, the labeled data is used to label the rest of the unlabeled data.
  • Great for large datasets such as webpages, videos, and audio files

Reinforcement Learning

  • Repetitive teaching
  • Kind of like giving the machines a test as the machine learns from feedback when it provides an incorrect answer.
  • Let’s say there’s a situation in which the machine has to guess what kind of music category is being played. If classical piano music is playing, but the machine says it’s “pop”, then the machine will be told it’s wrong. If it guesses “classical”, then the machine will be told that it’s right. The more data the machine is shown, the more the machine will know about the music category.

Challenges of Machine Learning

  • Lots of data is needed to start training a machine to understand your data.
  • Need to ensure that you have a good set of data to get the results you want.
  • Feature Engineering: cleaning up data to contain the features you want and getting rid of the features you don’t want
    • Feature selection: selecting features from data that you want to focus on
    • Feature extraction: using the features you collected already and continuing to modify it
    • Feature creation: when you get new data, you create new features

Before starting your own machine learning project

  • Determine your objective. Set the intention
  • Gather as much information as possible from as many sources as possible.
  • Data Collection - data wrangling, cleaning, pre-processing
  • Machine learning models
  • Keep restarting this process until you fulfill your objective
