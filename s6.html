What is Machine Learning?
Just like the name, the machines learn to process data and look for patterns. Then, based on those patterns, predictions can be made. 1959: Arthur Samuel

Impact of Machine Learning:
Machines have the ability to process lots of data at a time.
This makes them super efficient and provides a higher potential for accuracy.
More data → more accuracy, filter out data to narrow results down
More accuracy → better results

Application Examples:
Medicine: Using data from a variety of symptoms and the diagnosis for those range of symptoms, can make the diagnosis process more accurate and make sure we are as healthy as we can be.

Dating: Data from whether a certain profile is liked or disliked can lead to more accuracy in finding the love of your life.

Key Machine Learning Methods:
Supervised Learning
Unsupervised Learning
Reinforcement Learning

Supervised Learning:
Data is labeled. Less work for machines
Pros and Cons

Unsupervised Learning:
Data is NOT labeled. Machines look for patterns with culstering
Pros and Cons

Reinforcement Learning
Kind of like giving the machines a test. Let’s say there’s a situation in which the machine has to guess what kind of music category is being played. If classical piano music is playing, but the machine says it’s “pop”, then the machine will be told it’s wrong. If it guesses “classical”, then the machine will be told that it’s right. The more data the machine is shown, the more the machine will know about the music category.
Pros and Cons
