Session 8: Machine Learning Research Trends

Applications of Machine Learning in Various Fields

  • Analysis of stock trades, business risks, fraud, automatically manage portfolios
  • Criminal activity detection
  • Speech recognition use in virtual assistants such as Siri and Alexa
  • Maintain large amounts of data, such as hospital records, Google, YouTube videos
  • Speedy medical imaging analysis
  • Some examples:
    • Deep learning has produced incredible results, even going beyond human capability in nearly every field such as natural language processing, gaming, and speech recognition
    • Convolutional neural networks are skilled in classifying and analyzing patterns to recognize objects using vision
    • Recurrent neural networks is excellent in analyzing sequential data, and therefore is highly useful in improving speech recognition applications

Full-Stack Deep Learning

  • Full-stack deep learning is when you already have a deep learning model but still need to implement it into something, such as a mobile application.
  • Essentially, it’s the complete package, the full product that can be used to fulfill a purpose
  • The use of deep learning in businesses has led to a growing need for full-stack deep learning

Natural Language Processing

  • Natural language processing uses the computer to analyze human language and generate additional human language with the analysis
  • Can translate multiple languages, such as English to Chinese and vice versa
  • Spam filter
  • Google Search predictions
  • Auto-Correct in text messages

Natural Disaster Warning

  • Improving Earthquake Early Warning systems using machine learning using Distributed Multi-Sensor Earthquake Early Warning
  • The Earthquake Early Warning systems, which uses seismometers and GPS stations don’t produce very accurate data, but this development of Distributed Multi-Sensor Earthquake Early Warning uses machine learning for predictions, representation, and is able to process loads of data at once, giving extremely accurate results

Gaming

  • Machine learning in gaming is still a work-in-progress but with the increasing GPU processing speed and data available, there’s lots of potential with gaming.
  • Typically, games are hand-scripted but with machine learning, the possibilities can be completely unexpected and even the game designers might be surprised with what happens during the game
  • With so much data available, not only can there be several different outcomes in a game, the design can also look super realistic.
